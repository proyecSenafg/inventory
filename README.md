# inventory

**Proyecto inventory**

Gestor y administrador de ventas e inventarios para
ferretería.

> **Instrucciones de uso**

1. Ingrese a
   http://localhost/phpmyadmin/
2. Cree una base de
   datos y llámela pos.
3. Seleccione la
   base de datos pos y haga clic en la pestaña importar.
